Church Management System

A simple PHP & MySQL Church Management System developed by TM Lutere.

This system helps a church to manage:

- Members
- Events
- Registrations
- Admin dashboard

---

Requirements

- XAMPP (Apache & MySQL)
- PHP 8+
- Web browser (Chrome/Edge)

---

Installation Guide

1. Download the Project

Download or clone this repository into:

C:\xampp\htdocs\

After extraction you should have:

C:\xampp\htdocs\church_system

---

2. Start Server

Open XAMPP Control Panel and start:

- Apache
- MySQL

---

3. Create Database

1. Open browser and go to:

http://localhost/phpmyadmin

2. Click New
3. Create database named:

church_db

4. Click the database â†’ Import
5. Select the file:

church_db.sql

6. Click Go

---

4. Run the System

Open browser:

http://localhost/church_system/public
